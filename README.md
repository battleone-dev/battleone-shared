# Responsive Demo Site

Lightweight responsive starter built with HTML, CSS, and vanilla JS.

Preview locally:

```bash
# From the project folder
cd ~/responsive-site
# Python 3 built-in server on macOS
python3 -m http.server 8000
# Then open http://localhost:8000 in a browser
```

Files created:

- `index.html` — main markup
- `styles.css` — mobile-first styles and media queries
- `script.js` — small nav toggle script

Customize the layout, colors, and content as needed.
